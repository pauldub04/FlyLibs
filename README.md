## Installation
You need to install [Node js](https://nodejs.org/en/download/) and [npm](https://www.npmjs.com/get-npm)

### Run backend:
```bash
    cd node_backend
    npm install
    node index.js
```
or with nodemon for auto restaring server
```bash
    nodemon index.js
```

### Run frontend:
```bash
    cd vue_frontend
    npm install
    npm run serve
```
